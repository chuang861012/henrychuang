擔任研究助理期間建置與維護的教授個人網站。

### 架構

-   前端 : [Gatsby.js](https://www.gatsbyjs.com/)
    基於 Node.js 與 React 和 GraphQL 的靜態網站生成器。程式碼以 react 撰寫。
-   內容管理系統 : [Contentful](https://www.contentful.com/)
    雲端無頭CMS平台。
